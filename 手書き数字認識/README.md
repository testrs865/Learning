## Overview
- Pytorchを用いて手書き数字の10~15を識別するCNNモデルを実装しました
- AI技術の理解を深めるため、ChatGPTを活用しながらプロジェクトを作成しました

## Dataset
datasetの10~15はMNISTの0~5を組み合わせて作成しました。

## 概要
- このアプリケーションは、CSVファイルを読み込み、GUI上で内容を確認・編集・保存できるツールです
- コマンドライン操作が不要なためCSVを扱う初心者でも直感的に操作できます

---

## 制作目的

- Pythonの基礎文法（条件分岐・ループ・関数）を、実際に動作するアプリケーションの中で理解・定着させるため
- CSVファイルの仕様を理解し、業務データを安全に扱うための基礎を身につけるため
- 入力ミスや不正データに対するエラーハンドリングを実装し、
  実務を想定した堅牢な処理を意識するため
- 学習目的として、CUIでの処理を理解した上でGUI化し、保守性・拡張性を意識したプログラム構成を体験するため

※ 学習目的で作成したコードです。  
可読性と処理の流れが分かりやすい設計を重視しています。

---

## 主な機能

- CSVファイルの読み込み
- レコード単位での表示
- フィールド単位での表示
- 任意のレコード・フィールドの編集
- 編集内容の保存
- PySide6 Designer による UI 設計

---

## 使用技術

- Python 3.11.14
- PySide6
- Qt Designer
- NumPy（一部データ処理）
- Git / GitHub

---

## 実行方法
1. pip install PySide6 numpyをコマンドプロンプト(ターミナル)にて実行し、PySide6, numpyをインストールしてください
2. 本リポジトリをクローンします
3. config.pyにて**BASE_DIR**をmacOSなら**Path.home()** に、windowsなら**C:** にしてください
4. main.pyを実行してください

---

## 工夫した点
- CSVの仕様理解を深める目的で、標準の csv モジュールを使用せず、
  ファイルを1文字ずつ読み込みながら独自に解析処理を実装しました
- フィールド内のダブルクォーテーション数をカウントすることで、
  不正なCSV形式を検出できるようにしました
- 入力ミスがあった場合でもプログラムが異常終了しないよう、
  ループ制御と例外処理を組み合わせて実装しました
- データ構造を2次元リストで統一し、
  処理の流れが追いやすい構成を意識しました
- PySide6を使用してGUIを導入し、CSV編集や閲覧を直感的に操作できるようにしました
- 入力ミスや不正データ時のエラーメッセージをUI上で表示することで、ユーザーの操作性を向上

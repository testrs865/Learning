## CSV形式について
- 「Comma Separated Values」の略で、表形式のデータをカンマ（,）で区切って保存するテキストファイル形式です

## 表の1行が1つのレコードで表されます
- このアプリケーションでは次のように出力されます

例：
```
入力：
a,b,c
d,e,f

出力：
[レコード1]
    フィールド1 : a
    フィールド2 : b
    フィールド3 : c
[レコード2]
    フィールド1 : d
    フィールド2 : e
    フィールド3 : f
```

## 行内の格データ項目をフィールドと呼び、カンマ(,)で区切られています
- ただし、フィールド内にカンマ(,)を記述したい場合はダブルクォーテーション(")で囲む必要があります
- このアプリケーションでは次のように出力されます

例：
```
入力：
"a,a",b,c
d,e,f

出力
[フィールド1]
    レコード1 : a,a
    レコード2 : d
[フィールド2]
    レコード1 : b
    レコード2 : e
[フィールド3]
    レコード1 : c
    レコード2 : f

```

## フィールド内に「"」を記述したい場合は、「””」のように2回続けて記述します
例：
```
入力：
aaa,b""b,ccc
```
レコード1：aaa b"b ccc

## フィールド内に「"」が奇数個ある場合エラーです
例：
```csv
""aaa",bbb,ccc
```

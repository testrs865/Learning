# 📈 移動平均クロス戦略による株価分析ツール

## ■ 概要

本プロジェクトは、株価データを取得し、移動平均線を用いた売買シグナルの可視化および簡易的な分析を行うPythonツールです。
特に、ゴールデンクロスを利用した売買タイミングの把握に焦点を当てています。

---

## ■ 背景・目的

株式市場におけるトレンド分析の基本手法である移動平均線を用いて、
シンプルな売買戦略がどのように機能するかを検証することを目的としています。

---

## ■ 使用技術

* Python
* yfinance
* matplotlib

---

## ■ 機能

* 株価データの取得（Yahoo Finance API）
* 移動平均線の計算（5日・25日）
* ゴールデンクロスの検出
* 売買シグナルの可視化
* 出来高の表示（別軸）

---

## ■ ゴールデンクロス戦略について

短期移動平均線（5日）が長期移動平均線（25日）を上抜けるゴールデンクロスを、
上昇トレンドの初動と捉えて買いシグナルとして実装しました。

このシグナルにより、トレンド転換のタイミングを視覚的に把握することが可能です。

ただし、レンジ相場などではダマシ（誤シグナル）も多く発生するため、
今後は他の指標との組み合わせによる精度向上を検討しています。

---

## ■ 実行例

* 銘柄コード：4755.T
* 期間：2023-01-01 〜 2024-01-01

出力内容：

* 株価チャート
* 移動平均線（MA5 / MA25）
* ゴールデンクロス（▲表示）
* 出来高
